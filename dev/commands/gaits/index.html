Gaits

Gaits are continuous locomotion patterns: the robot keeps moving in the requested manner until you send a different command (for example a posture such as PetoiBittle.balance to stop and stand). Each gait comes in forward / left / right variants where the firmware supports them.

Use case: walk around, then stop

using PetoiBittle

connection = PetoiBittle.connect(PetoiBittle.find_bittle_port())

PetoiBittle.walk_forward(connection)   # start walking forward
sleep(3)                               # let it walk for a few seconds
PetoiBittle.walk_left(connection)      # curve to the left
sleep(3)
PetoiBittle.balance(connection)        # stop and stand still

PetoiBittle.disconnect(connection)
Tip

Trotting is faster than walking; crawling keeps the body lower and more stable. Pick the gait that matches your surface and speed needs.

WalkForward()

A PetoiBittle.Command (category: Gaits) that makes the robot walk forward.

Serializes to the firmware token "kwkF". Send it explicitly with PetoiBittle.send_command, or use the convenience verb PetoiBittle.walk_forward.
